# 買取一丁目 JAN価格チェッカー

Tailwind CSSを使ったシンプルで美しいWebアプリです。

JANコードを入力すると、買取一丁目の買取価格を参照（デモデータ）します。

## 使い方

1. JANコード（8桁または13桁）を入力
2. 「価格を検索」ボタンをクリック
3. 結果が表示されます

## 技術スタック

- HTML5 + Tailwind CSS (Play CDN)
- Vanilla JavaScript

## GitHub Pagesでの公開

このリポジトリはGitHub Pagesで公開可能です。

**Settings > Pages > Source: Deploy from a branch > Branch: `main` / Folder: `/ (root)`**

## 注意事項

- これはデモアプリケーションです。実際の価格は[公式サイト](https://www.1-chome.com/)でご確認ください。
- 価格は市場状況により変動します。
